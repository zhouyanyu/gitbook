### 简介 <a href="#jia" id="jia"></a>

弩手是更强大的轻型射手单位，他们相比于步弓手射速更慢，但是他们发射的高穿透力的弩箭可以对重装部队造成严重威胁\
弩手是城堡时代(III)生产的部队，没有任何文明可以获得早期版本的弩手\
马里人独特的掷枪兵取代了他们的弩手，尽管性质上不太一样
