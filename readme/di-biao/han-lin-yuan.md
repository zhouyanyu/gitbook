### 简介 <a href="#jia" id="jia"></a>

翰林院是中国人升级到时代2的两个地标之一，它可以使得自身影响力范围内的建筑物生产的税金加倍，同时可以在这里解锁朝廷命官的加倍收税效率科技\
在11963补丁后，现在朝廷命官可以把税上缴给翰林院，并且在第二季更新后解锁朝廷命官的生产功能
