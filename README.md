## 简介

中国是帝国时代4原版首发的8个文明之一，他们是这个游戏最难掌握的文明之一\
\
游戏里的中国人基于唐朝灭亡后混乱的五代十国时期直到明朝灭亡时期(907-1644)在中原大地上建立多个王朝的中华民族，即使经历了无数的改朝换代和战乱动荡，中国人依旧保有着自身独特的文化和创造力在中世纪时期闪闪发光，这在他们历经四个朝代所说的古汉语中可以得到体现，并且他们发明出来的火药将会永远地改变人类在之后的作战方式\
\
中国人发明了一套高效的行政系统，可以确保这个巨大的帝国可以有条不紊地运作下去，朝廷命官是帝国法则的执行者，他们是经过了科举考试选拔的最有才华的文人，组成了这台巨大的机器的核心部分：官员不仅可以通过监督资源建筑使得文明可以获得更多的资源，还可以监督科技建筑和军事建筑大幅度地提高它们的效率，同时玩家的各个行为都会产生税金，朝廷命官还有一个作用就是孜孜不倦地从建筑中回收税金来补充帝国的国库，如何让文明迅速蓬勃发展完全取决于玩家对官员的理解和调配\
\
中华文明最大的特征就是从数千年前就开始延续不断的王朝，即使王朝在动荡和衰落中走向了终结，一个新的王朝总会再次崛起，并制定新的秩序，这体现于他们在游戏里的独特的王朝系统，中国人可以建造同时代的两个地标来进入新的王朝并获得这个王朝独特的加成和部队，使得他们可以适应各种各样的恶劣环境，策划不同的战略来让文明抵御来自不同的敌人的冲击\
\
从秦汉时期，来自漠北的游牧民族就不断地侵入中华帝国的核心区域，中国人在帝国的北部建造了延绵万里的伟大防御设施——长城，这也是游戏里的中国人是强大的防御型文明的原因，中国人的城墙可以被时代4的地标长城门楼强化变得更加坚固，同时独特科技“额外材料”会使得大型箭塔附近的城墙可以缓慢地恢复生命值，更好地抵御敌人的入侵\
\
中国人是勤劳而富有创造力的民族，建造任何设施中国人都可以以两倍的速度更快地建造完毕，配合其强大的城市防御能力，使得中国人的奇观胜利更加可靠，同时中国人也是中世纪最早将火药应用于战争中的民族，早在宋朝时期，中国人就尝试使用火药封装在罐子内投掷引爆，或是在长枪的枪头旁边设置点燃的炸药，在元朝时期，最原始的火枪和火炮开始出现，明朝时期更是出现了强大的使用火药动力推进的箭矢发射器，这体现于游戏内各种五花八门的火器独特部队和加成科技中

## 精通项目历史文本 <a href="#his" id="his"></a>

独特的建筑\
对创造出巨大内部空间的渴望带来了建筑技术的革新，进而使中国传统建筑有了独特的造型。名为“斗栱”的榫接系统分散了屋梁的重量，从而减少了内部所需的支柱。通过这种方式，房间就能以轻巧的织物进行分隔，甚至完全不加隔断。\
\
纸币\
随着中国宋朝商业的繁荣发展，商人及税官不得不携带越来越重的大批硬币。但在印刷技术改进后，世界上最早的纸币也就此问世。虽然因为重量较轻，解决了搬运时的问题，但容易制造的特性很快就导致了通货膨胀及伪造等其他问题。\
\
穆桂英\
据说，北宋的传奇将军穆桂英之所以被称为精明的军事领袖，是因为她发现了击破“天门阵”的方法。她没有正面进攻这个坚不可摧的阵型，而是摧毁了敌方的补给，使敌人很快就陷入了饥荒并投降。她也因此获得了朝廷的嘉奖：允许她挑选自己的丈夫。\
\
连弩\
在村庄遭受袭击时，中世纪的中国农民会使用一种巧妙的武器来抵御进犯。连弩可以在敌人接近时快速射出一连串淬毒的弩箭，虽然射程较短且威力不大，但压倒性的数量已经足以击退入侵者。\
\
城镇规划\
中国古代都市都经过细心规划，构成了今日所谓的棋盘式格局。与现代都市相反的是，当时的公共建筑设在郊区，而私人住宅则是设在城镇中心。此外，由同心圆式的城墙及数座城门构成的城市体系可避免没有获得适当许可的人进入内城的核心区域。\
\
士大夫\
朝廷士大夫是一个显耀且尊贵的职位，皇帝会亲自从有资格的人选中任命这个官职。这种令众人艳羡的声望也导致中国的家族之间产生了明显的社会阶级差异，大量富裕的精英家族会对男性子孙进行培养，只为参加一项重要的事件：参加科举考试。\
\
三十六计\
中国古代的谚语经常被应用到军事策略当中，因为这些谚语在战争中能为将领提供睿智的思维。三十六计的内容包括如何打垮弱小的敌人 (关门捉贼)，或是将强大的敌人引诱到弱势的位置 (调虎离山)。而如果一切计谋都失败了，第三十六计则提出了撤退的建议：“走为上策”。\
\
珍贵作物\
在宋朝，人们培育出能迅速成熟且耐旱的新品种稻米。这种快速成熟的稻米只需数个月就能收割，因此在一年当中就可以种植并收割第二期的作物。随着作物产量的蓬勃发展，华中地区的势力也随之增强。\
\
传奇之刃\
偃月刀，又称关刀，是中国传统的长柄兵器。这种刀的地位被人们神化了，据说一位中国的传奇将军发明了这件武器，而且他能轻而易举地驾驭这把沉重无比的大刀。经年累月过去，这种武器的刀刃变得越来越重，而具备掌控这把武器的能力，则成为了一项参军时的艰难考验。\
\
天文钟\
宋朝科学家使得中国在诠释天文原理方面有了飞跃性的成果。这些知识搭配上工程科技的进步，造就了一座位于开封的外型宏伟的水力天文钟楼。这座钟楼不但可以显示时间、日期、月相，甚至还能显示行星的位置。\
\
火焰阵\
除了许多出其不意的战略之外，中国人有时也会使用不寻常的方式来运送火焰并使敌人着火。他们会在活生生的动物 (例如鸟或牛) 身上绑上火球或是能延后引燃的燃烧弹，然后将这些动物送入敌人的领地。这些动物造成的混乱及破坏可以削弱敌人，制造出适合发动攻击的时机。\
\
第一把火器\
中国的火长矛是所有火器的前身。在发现火药后，火焰投掷器及爆炸性武器很快就随之诞生，让与中国人为敌的对手在战场上胆战心惊。火长矛据说是蒙古人唯一真正畏惧的中国武器。\
\
宝塔传说\
人们认为宋代的宝塔充满着超自然的力量，但早期的木造宝塔往往无法抵御自然灾害的侵袭，因此人们认为宝塔的毁灭是上天对人类近期犯下的恶行所降下的惩罚。这些宝塔之后通常会以石料重建，因为人们希望结构更坚固的宝塔能带来更好的运气。\
\
以火防御\
中国人不只会用燃烧武器来攻击敌人，还能用它来保卫城市。他们会在牵引式的巨型投石机上装填带有倒勾的火球，这些长满尖刺的火焰炮弹会勾住敌人攻城塔的木支柱并持续燃烧，运气好的话，可以一直燃烧到攻城塔崩塌为止。\
\
幼帝\
在维持了三个多世纪的统治之后，南宋在蒙古人手中迎来了末日。在维持王朝存续的迫切需求下，年仅七岁的赵昺被拥立成为皇帝，但他注定悲剧的统治仅仅持续不到一年。蒙古人的追捕让他早早就丧了命。
